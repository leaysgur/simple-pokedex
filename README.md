# simple-pokedex
レートに潜るにしてはあまりに無知すぎる自分のために、サクッとデータをひける簡易ポケモンずかんをつくった。

## 使ったものメモ
- Backbone.js
- Backbone.Marionette.js
- Require.js

詳細は[ブログ](http://lealog.hateblo.jp/)で書いてるかも。

## もし動かしてみるなら
```
git clone git@github.com:leader22/simple-pokedex.git
npm install
grunt build
```
